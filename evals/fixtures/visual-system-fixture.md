# Palette
- Background hex: #0F1115
- Primary accent hex: #22D3EE
- Secondary accent hex: #F472B6
- Neutral hex: #E5E7EB

# Typography
- Headline weight: 800, case: lowercase
- Body weight: 500, case: sentence
- Pull-quote treatment: 600, italic, accent color

# Layout
- Slide arc: HOOK → REVEAL → SETUP → EXAMPLES → OUTCOME → CTA
- Negative space rules: 18% min margin, headline upper-third
- Icon style: line, 2px stroke, no fills

# Vibe Rules
- Tone: editorial, candid
- What to avoid: stock photography, Canva overlays, gradient meshes, emoji clusters

# Output sizes
- LinkedIn carousel: 1080x1350
- IG/LI feed: 1080x1080
- Stories/Reels: 1080x1920
