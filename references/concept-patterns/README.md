# Concept patterns

Kit-shipped exemplar patterns derived from Matt's published carousel work.
The concept-generator skill loads these to inform Stage 1 (concept
generation): each pattern shows what a high-saving concept looks like,
which format it tends to fit, and what hook style anchors it.

When operators provide their own annotated reference corpus (see
`references/carousel-examples/`), the skill loads both. Brand-specific
patterns layer on top of these defaults.

## The six exemplars

- `stack_contrast.md`: stack vs stack ("$11k photographer" vs "one prompt")
- `agency_anchor.md`: agency invoice line item as receipt anchor
- `dated_personal_anecdote.md`: tuesday-2pm style dated personal scene
- `named_framework_close.md`: 2-3 word framework name that becomes the save trigger
- `borrowed_authority.md`: quote a known operator/founder, then contextualize
- `insider_observation.md`: point at something the reader sees but hasn't named

Each file follows the same shape: hook style, format hint, body density
markers, close style, why it saves.
